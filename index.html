TypeCode Todos

A Todo Application

  • Enter a todo in the input field and click the add button
  • Click on the todo to mark it as completed
  • Click on the corss icon to delete the todo
  • Use the search bar to filter todos
  • Click on the clear button to clear all completed todos
  • Click on the all button to show all todos
  • Click on the active button to show active todos
  • To persist the data here the browser local Stroage has been used
  • This has been develop using the ES6 vanilla JS
